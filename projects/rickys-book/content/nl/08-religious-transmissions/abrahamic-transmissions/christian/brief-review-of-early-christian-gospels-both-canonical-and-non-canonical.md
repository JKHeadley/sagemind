# Kort Overzicht van Vroeg-Christelijke Evangeliën, zowel Canoniek als Niet-Canoniek



|
Evangelienaam
 |
Primair Publiek
 |
Kernkenmerken
 |
Belangrijkste Thema's
 |  |
|
Marcus (canoniek)
 |
Romeinse of vervolgde heidense christenen
 |
Kortste; urgente toon; actiegedreven; weinig leringen; abrupt einde
 |
Lijdende Messias, geheimhoudingsmotief (Messiaans geheim), navolging door beproeving
 |  |
|
Mattheüs (canoniek)
 |
Joodse christenen
 |
Gestructureerd als Torah (5 redevoeringen); veel OT-citaten; benadrukt profetie
 |
Jezus als Nieuwe Mozes, vervulling van profetie, Koninkrijksethiek
 |  |
|
Lucas (canoniek)
 |
Hellenistische heidense christenen, gemarginaliseerden
 |
Langste evangelie; gepolijst Grieks; universele toon; vrouwen, armen, uitgestotenen
 |
Universele verlossing, mededogen, Heilige Geest, geschiedenis van de Kerk
 |  |
|
Johannes (canoniek)
 |
Gehelleniseerde joodse en heidense christenen
 |
Poëtisch, symbolisch; niet synoptisch; tekenen en "Ik ben"-uitspraken; kosmische proloog
 |
Goddelijkheid van Jezus, eeuwig leven, dualisme (licht/duisternis), liefde
 |  |
|
Evangelie van Thomas
 |
Mystieke/gnostisch georiënteerde christenen
 |
114 uitspraken van Jezus; geen narratief; geheime kennis
 |
Innerlijke verlichting, zelfkennis, koninkrijk binnenin
 |  |
|
Evangelie van Maria
 |
Gnostische christenen; mogelijk vrouwelijke leiders
 |
Fragmentarisch; dialoog na opstanding; Maria als lerares
 |
Spirituele opstanding, innerlijke waarheid, kritiek op Petrus' autoriteit
 |  |
|
Evangelie van Petrus
 |
Apocalyptische christenen (mogelijk anti-joods)
 |
Lijdensverhaal; dramatische opstandingsscène; Jezus lijdt niet
 |
Triomf over de dood, kosmisch conflict, anti-judaïsche toon
 |  |
|
Evangelie van Judas
 |
Sethiaanse Gnostische sekte
 |
Dialoog tussen Jezus en Judas; Judas als verlichte
 |
Geheime kosmologie, kritiek op kerkleiders, verraad als gehoorzaamheid
 |  |
|
Kindheidsevangelie van Jakobus
 |
Vroege Maria-vereerders
 |
Richt zich op Maria's wonderbaarlijke geboorte en Jezus' kindertijd
 |
Reinheid, goddelijke interventie, maagdelijkheid van Maria
 |  |
|
Evangelie van de Hebreeën
 |
Joods-christelijke sekte (bijv. Ebionieten)
 |
Alleen bekend via fragmenten/citaten van Kerkvaders
 |
Benadrukte mogelijk Jezus' menselijkheid en gehoorzaamheid aan de Wet
 |  |
