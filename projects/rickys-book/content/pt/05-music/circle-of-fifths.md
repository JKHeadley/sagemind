# Ciclo das Quintas



A Regra das Quintas — frequentemente chamada de Ciclo das Quintas — é um conceito fundacional na teoria musical que revela as relações entre os 12 tons da escala cromática, suas armaduras de clave correspondentes e como as tonalidades se relacionam harmonicamente. É essencial para compreender harmonia, modulação e composição.



1. O que é uma "Quinta"?

Uma quinta justa é um intervalo que abrange cinco notas em uma escala diatônica. Por exemplo:

	•	De Dó a Sol é uma quinta justa:

	•	C–D–E–F–G → 5 graus da escala.



Este intervalo tem uma razão de frequência de 3:2, tornando-o um dos intervalos mais consonantes depois da oitava.

4. Equivalentes Enarmônicos

	•	Si maior = Dób maior

	•	Fá# maior = Solb maior

	•	Dó# maior = Réb maior

Essas tonalidades soam iguais mas são escritas de formas diferentes dependendo do contexto.



⸻

5. Usos da Regra das Quintas

a. Memorização de Armaduras de Clave

Ajuda a determinar rapidamente quantos sustenidos ou bemóis uma tonalidade tem.

b. Modulação

Modular para tonalidades uma quinta distante soa natural e suave. Modulações comuns:

	•	Dó maior → Sol maior (subindo uma quinta)

	•	Lá menor → Mi menor (subindo uma quinta)



Modulação para tonalidades proximamente relacionadas (dentro de 1–2 passos no ciclo) mantém coerência harmônica.

c. Progressões de Acordes

Progressões baseadas em quintas soam fortes e direcionais. Exemplos comuns:

	•	Progressão do ciclo de quintas: D7 → G7 → C7 → F7 → Bb7

	•	Esta é a base para turnarounds, progressões de jazz e sequências barrocas.



d. Função Harmônica e Tensão

Quintas descendentes (ex.: V → I) resolvem a tensão efetivamente. Por exemplo:

	•	G7 → Dó maior (V7 → I)



Esta é a função cadencial e é central para a harmonia funcional.



6. Ciclo das Quintas na Composição

	•	Na composição clássica, compositores frequentemente se movem ao redor do ciclo para criar tensão narrativa.

	•	No jazz, a progressão ii–V–I frequentemente percorre partes do ciclo.

	•	No pop e rock, a progressão I–V–vi–IV (C–G–Am–F) usa relações adjacentes no ciclo para produzir ressonância emocional.



7. Representação Visual

Imagine um relógio:

	•	12 horas: Dó maior

	•	1 hora: Sol maior (1 sustenido)

	•	2 horas: Ré maior (2 sustenidos)

	•	… etc., movendo no sentido horário adiciona sustenidos

	•	Movendo no sentido anti-horário adiciona bemóis.



Na parte inferior (6 horas), Fá# maior e Solb maior se encontram.



8. Avançado: Mediante Cromática e Relações de Tonalidades Distantes

	•	Tonalidades que não são adjacentes no ciclo têm menos notas em comum.

	•	Modular para estas cria contraste dramático (ex.: Dó maior para Mi maior).
